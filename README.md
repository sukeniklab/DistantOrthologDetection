# Distant Ortholog Detection pipeline associated wtih "A geothermal amoeba sets a new upper temperature limit for eukaryotes" #

Ortholog detection pipeline from the manuscript available at [https://www.biorxiv.org/content/10.1101/2025.11.24.690213v1](https://www.biorxiv.org/content/10.1101/2025.11.24.690213v1)

Repository Structure: 
```
.
├── 1A_EmbedSequences
├── 1B_Orthofinder
├── 2_ReciporcalBestHits
├── 3_CombinePotentialOrthologs
└── 4_FoldseekFiltering

```
Directories within this repository are numbered in order in which they should be completed.
